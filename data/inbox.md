# Inbox — links e temas para próxima edição

Este arquivo acumula submissões enviadas para `diariaeditor@gmail.com` durante o dia. O orquestrador o lê no início do Stage 1 e arquiva em `data/inbox-archive/{YYYY-MM-DD}.md` após aprovação humana.

Entradas são adicionadas automaticamente por `inbox-drainer`. Também pode ser editado manualmente.

Formato de cada entrada:

```
## {timestamp ISO}
- **from:** {remetente}
- **subject:** {assunto}
- **urls:** [url1, url2, ...]  (opcional)
- **topic:** {texto livre se não houver URL}  (opcional)
- **raw:** {trecho do corpo do e-mail}
```
